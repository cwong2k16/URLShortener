Hi! This is my short URL microservice. Here is how you can use it:

Original URL

You can enter an original, HTTPS url at the end of this link following "/api/", like this:

localhost:3005/api/https://wwww.google.com/

This will generate you back a short URL which links you to https://www.google.com/

Short URL

This is the short URL:

localhost:3005/url/3741

Pay special attention! The short URL is a 4 digit number followed by "/url/", not "/api/".
